## Installing House of Electronics Sales Manager on macOS

### **Step 1: Download the App**
- Download the `House of Electronics Sales Manager-1.0.0.dmg` file
- Double-click the DMG file to open it

### **Step 2: Install the App**
- Drag the "House of Electronics Sales Manager" app to the "Applications" folder
- Wait for the copy to complete
- Eject the DMG file

### **Step 3: First Launch (Important!)**
When you first try to open the app, macOS will show a security warning. This is normal for apps not from the App Store.

**To open the app:**

1. **Right-click** on the app in Applications folder
2. Select **"Open"** from the context menu
3. Click **"Open"** in the security dialog

**Alternative method:**
1. Go to **System Preferences** > **Security & Privacy**
2. Click the **"Open Anyway"** button next to the blocked app
3. Click **"Open"** to confirm

### **Step 4: License Activation**
1. The app will prompt you to activate your license
2. Copy your Machine ID and send it to your vendor
3. Import the license file when you receive it

### **Troubleshooting**

**"App is damaged and can't be opened"**
- This is a false positive from macOS
- Right-click the app and select "Open"
- Or use the System Preferences method above

**"App can't be opened because it is from an unidentified developer"**
- This is normal for non-App Store apps
- Use the right-click "Open" method

**App won't start after installation**
- Make sure you're using the right-click "Open" method
- Check that you have macOS 10.15 or later

### **System Requirements**
- macOS 10.15 (Catalina) or later
- 4GB RAM minimum
- 100MB free disk space
