## Installing House of Electronics Sales Manager on Windows

### **System Requirements**
- Windows 10 or later (64-bit recommended)
- 4GB RAM minimum
- 200MB free disk space
- Internet connection for license activation

### **Step 1: Download the Installer**
- Download the `House of Electronics Sales Manager-1.0.0-Setup.exe` file
- Save it to your Downloads folder or Desktop

### **Step 2: Run the Installer**
1. **Double-click** the installer file
2. If Windows shows a security warning, click **"More info"** then **"Run anyway"**
3. The installation wizard will start

### **Step 3: Installation Process**
The installer will guide you through these steps:

1. **Welcome Screen** - Click "Next" to begin
2. **License Agreement** - Read and accept the terms
3. **Installation Directory** - Choose where to install (default is recommended)
4. **Installation Progress** - Watch the detailed installation steps:
   - Installing application files
   - Setting up license system
   - Creating shortcuts
   - Configuring security settings
5. **Completion** - Choose whether to launch the app immediately

### **Step 4: First Launch**
1. The app will start automatically (if you chose to launch it)
2. You'll see the license activation screen
3. Copy your Machine ID and send it to your vendor
4. Import the license file when you receive it

### **What Gets Installed**
- **Application Files** - Main program in Program Files
- **License System** - Secure license validation
- **Desktop Shortcut** - Easy access from desktop
- **Start Menu Entry** - Available in Windows Start Menu
- **Uninstaller** - Clean removal option

### **Troubleshooting**

**"Windows protected your PC" warning**
- Click "More info"
- Click "Run anyway"
- This is normal for non-Microsoft Store apps

**Installation fails or stops**
- Run as Administrator (right-click â†’ "Run as administrator")
- Check you have enough disk space
- Temporarily disable antivirus if needed

**App won't start after installation**
- Check Windows Defender isn't blocking it
- Try running as Administrator
- Restart your computer

**License activation issues**
- Make sure you have internet connection
- Check your Machine ID is correct
- Contact support if problems persist

### **Uninstalling the Software**
1. Go to **Settings** > **Apps** > **Apps & features**
2. Find "House of Electronics Sales Manager"
3. Click **"Uninstall"**
4. Choose whether to keep your license data
